FDM Printing

First I had to measure the circle in the paint organizer. It was 3.5 inches in diameter with a .24 inch gap.


Next I started to create my planter on Fusion 360 I built the base using the measurements of the circle to make sure that it fits inside. The base was made of a 3.5 inch diameter outer circle, then using the offset tool i created another circle inside with a -.232 gap


I then used the extrude tool to build my base, I put the edge of the circle down a quarter of an inch and the inner part up about a fifth of an inch


After building the base I had to make two offset planes putting one at 1.5 inches and the other at 2.5 inches.I then had to decide what shape I wanted my planter to be. I placed two polygons, one on each offset plane. I made the lower one slightly bigger than the higher one.


Using the loft tool I created a shape in between the two planes and the base. Then I had to design a pattern that would go on the outside of my pot. After some debate I decided on trapezoids. here is final design


After using the shell tool to hollow out my design. I had to save it as an STL file and then opened it in Prusa slicer.

Prusa slicer is a program that is used in conguction with Prusa brand 3d printers. It reads stl files and allows you to see all the details of your print. Including all the layers of it and the time it will take to print. It will also allow you to add supports (extra filiment used to stop the print from caving in) Because my print had a lot of different hollow segments I needed supports everywhere. After looking at my print and deciding it was printable, I saved the file as gcode and put it on an SD card


After cleaning the print bed, apply isoprobal alcohol to the bed, and changing the color of the filiment (PLA), it was time to put the SD card in and select my file and start printing the pot. the first two attempts failed in the very early stages because there was no glue on the bed. The third attempt faild because the machine ran out of filiment. The fourth one was able to print all the way through
