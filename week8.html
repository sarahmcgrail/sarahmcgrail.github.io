Intergration

The project this week was to create a laser cut acrylic light up sign with a 3d printed base. This project was really fun and the end product is very cool.


My first thought for designing the design for the sign was what would look good in my house, but also would make my parents say aww when they saw it. So my first idea was to do something surronding our cats and it snowballed from there into all the cats that my parents have ever owned together. In Adobe illustrator I just started drawing pawprints until I got to a design that I liked then I copied it ten times, putting the name of a cat under each one. file design:https://drive.google.com/file/d/1XCSj3YGRNzycTs7BEvXx7meCHE832q99/view?usp=sharing Picture of test print on wood

Learning how to solder was fun, My dad was a little jealous. I was in the very first training so the board that I built wasn't used in the final design. The one in the final project was prebuilt for me.

I designed th 3d printed base by using the lasercut wood file as the top then making another slightly bigger rectangle an inch below then using the loft tool to connect the two then used the shell tool to hollow it and finally the emboss tool to cut the slot in the top. In my first print of the base I had forggotten to put a hole into it for the cord to go through, so i made another one with a hole in it.
